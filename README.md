# PreLoved Market  
### A Trusted Second-Hand Fashion Marketplace for Students

---

## Overview

PreLoved Market is a mobile application that enables students to buy and sell second-hand clothing within their school community.

This idea was developed through user research, with the goal of solving real challenges faced by students when accessing affordable and reliable fashion.

---

## Problem

Currently, students rely on informal platforms such as WhatsApp groups, street vendors, and social media. These methods present several challenges:

- Difficulty finding specific clothing items  
- No reliable way to verify size and fit  
- Poor-quality product images  
- Lack of trust and accountability between users  

---

## Solution

PreLoved Market provides a centralized and trusted platform where students can easily buy and sell second-hand clothing within their community.

Value Proposition:  
An easy and trusted marketplace for second-hand fashion.

---

## Core Features (MVP)

| Feature | Description |
|--------|------------|
| Product Listings | Users can post items with photos, size, price, and condition |
| Search & Filter | Users can find items by category, size, and price |
| Ratings & Reviews | Builds trust through seller feedback |
| In-App Chat | Enables direct communication between buyers and sellers |
| Size Guide | Helps users verify fit before purchasing |

---

## Tech Stack

- Frontend: Flutter  
- Database: SQLite  
- Design & Collaboration: Figma, GitHub  

---

## Key Constraints

- Limited Internet Access → Lightweight app design  
- Trust Issues → Ratings and reviews system  
- Device Limitations → Simple UI and local storage
